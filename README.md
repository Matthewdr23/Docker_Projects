# Docker Projects
### Purpose
This repository is a collection of Docker projects, ranging from web servers and database servers to various other applications. 
Its primary goal is to provide vulnerable-by-design containers, giving aspiring cybersecurity professionals the opportunity to gain hands-on experience with Docker, container security, and system hardening.
